符号链接

  • 时间：2025/1/4 10:08:36
  • 分类：??
  • 阅读时间：??min
  • 原创
“pnpm link” 是符号链接（symbolic link）。符号链接是一种特殊的文件类型，它指向另一个文件或目录的路径。在使用 “pnpm link” 时，它创建的符号链接允许在不同的项目之间快速共享和引用本地的包，而无需进行实际的复制操作。这种方式可以方便开发过程中的调试和代码复用。
