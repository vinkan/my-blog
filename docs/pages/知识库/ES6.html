数组 ​

concat() ​

连接两个或更多的数组，并返回结果。

  • 语法格式
js
array1.concat(array2, array3,..., arrayX)
1
  • 参数说明
参数描述
array2, array3,..., arrayX必需。需要连接的数组。
  • 示例
js
// a 与 b 数组合并
var array1 = ["小猫", "小狗"];
var array2 = ["小鸡", "小鸭", "小鹅"];
var array3 = array1.concat(array2);

// 输出 array3
["小猫","小狗","小鸡","小鸭","小鹅"]

copyWithin() ​

从数组的指定位置拷贝元素到数组的另一个指定位置中。

  • 语法格式
js
array.copyWithin(target, start, end)
1
  • 参数说明
参数描述
target必需。复制到指定目标索引位置。
start可选。元素复制的起始位置。
end可选。停止复制的索引位置 (默认为 array.length)。如果为负值，表示倒数。
  • 示例
js
const array = ["猫", "狗", "猪", "鸡", "鸭", "鹅"];
const res = array.copyWithin(2, 0, 2);

// 输出 res
 ['猫', '狗', '猫', '狗', '鸭', '鹅']

entries() ​

返回一个数组的迭代对象，该对象包含数组的键值对 (key/value)。

  • 语法格式
js
array.entries()
1
  • 示例
js
const array1 = ['猫','猴','狗','蛇']
const x = array1.entries()
const res1 = x.next().value
const res2 = x.next().value

// 输出 res1 res2
[0, '猫']  [1, '猴']

fill() ​

使用一个固定值来填充数组。

  • 语法格式
js
array.fill(value, start, end)
1
  • 参数说明
参数描述
value必需。填充的值。
start可选。开始填充位置。
end可选。停止填充位置 (默认为 array.length)
  • 示例
js
var fruits = ["喜羊羊", "美羊羊", "懒羊羊", "羊博士"];
const array = fruits.fill("灰太狼", 2, 3);

// 输出 array
['喜羊羊', '美羊羊', '灰太狼', '羊博士']

filter() ​

检测数值元素，并返回符合条件所有元素的数组。

  • 语法
js
array.filter(function(currentValue,index,arr), thisValue)
1
  • 参数说明
参数描述
function(currentValue, index,arr)必须。函数，数组中的每个元素都会执行这个函数
函数参数:
currentValue 必须。当前元素的值
index 可选。当前元素的索引值
arr 可选。当前元素属于的数组对象
thisValue可选。对象作为该执行回调时使用，传递给函数，用作 "this" 的值。如果省略了 thisValue ，"this" 的值为 "undefined"
  • 示例
js
// 遍历
const ages = [1,2,3,4,5,6,7,8,9];
const res = ages.filter((ele)=>ele%2===0);

// 输出 res
[2, 4, 6, 8]
